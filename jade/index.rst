The JADE Facility
=================

The JADE facility consists of 2 head nodes and 22 NVIDIA `DGX-1 <https://www.scan.co.uk/3xs/info/nvidia-dgx-1>`_ servers, each with 8 GPUs and 40 CPU cores. For the Hackathon event 2 nodes have been reserved for exclusive access.


**Official Documentation Site**

The official JADE documentation site is located at `jade.ac.uk <https://jade.ac.uk>`_
